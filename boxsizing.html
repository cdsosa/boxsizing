This box is 300 x 200px with 25px of padding. The padding adds to the height or width of the box; thus, the actual size of the box increases. Of course, the text does not invade the padded area.
This box is 300 x 200px with 25px of padding and 25px borders. The padding and borders add to the width and/or height of the box.
This box is 300 x 200px with 25px of padding and 25px borders AND uses box-sizing:border-box.

Box-sizing:border-box appled to a container, causes padding and borders to be included in the width and/or height of the box. Therefore, the outer dimension of the container do not change.
